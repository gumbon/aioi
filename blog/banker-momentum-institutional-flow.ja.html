バンカー vs リテーラー モメンタム：機関投資家フローの読み方

AIO Indicator Team April 14, 2026 13 min read

問題点：すべてのモメンタムが同じに見える

ほとんどのトレーダーは、すべての市場参加者を平等に扱う RSI、MACD、またはモメンタムオシレーターに頼っています。リテールのデイトレーダーと何十億ドルものヘッジファンドが同じ RSI の読み値に寄与しています。つまり、機関投資家の買い集めとリテールの FOMO を区別できません — これは重大な盲点です。

AIO Banker Momentum Volatility は、モメンタムを3つの異なる参加者層に分解することでこれを解決します。それぞれ異なる感度と時間軸を持ち、実際に市場を動かしているのが誰かを把握できます。

3層構造の RSI 分解

核心的なイノベーションは、RSI ベースのモメンタムを異なる市場参加者を表す3つの層に分割することです：

1. バンカーモメンタム（機関投資家）

  • RSI 期間： 50バー（遅い、ノイズをフィルタリング）
  • RSI ベース： 50（ニュートラルゾーン以上のモメンタムのみカウント）
  • 感度乗数： ×2（視認性のために増幅）
  • 計算式： 2.0 × max(0, (RSI(50) - 50) / (100 - 50)) × 100

この層は、機関投資家が数日から数週間にわたって買い集めることで生成される、ゆっくりとした持続的なモメンタムを捉えます。バンカー列が現れると、50期間 RSI が長期間にわたって50を上回っており、大手プレーヤーのみが維持できる継続的な買い圧力を示しています。

2. ホットマネー（投機的資本）

  • RSI 期間： 40バー
  • RSI ベース： 30（低い閾値 — より多くの活動をキャプチャ）
  • 感度乗数： ×0.7

“ホットマネー”は投機的資本を表します — ヘッジファンド、モメンタムトレーダー、プロップファームなど、素早く動く参加者です。機関投資家の後、リテールの前にエントリーし、トレンドの加速フェーズを生み出します。

3. リテーラーモメンタム

  • RSI 期間： 14バー（標準、速い）
  • RSI ベース： 20（非常に低い閾値 — すべての活動をキャプチャ）
  • 感度乗数： ×0.3（抑制）

最も速く、最もノイジーな層です。バンカーのサポートなしにリテーラーのみのモメンタムが見られる場合、通常は FOMO 主導の持続不可能な動きを示しています。

積み上げ列の読み方

3つの層はパーセンテージとして積み上げられます（バンカー% + ホットマネー% + リテーラー% = 100%）。高さと構成が示すものは：

  • バンカー（緑）が支配する高い列： 強い機関投資家の参加 — 継続動向の高確率
  • 75%以上の列（支配閾値）： 銀行が完全にコントロール — トレンドが継続する可能性が非常に高い
  • 50〜75%の列： バンカーは活発だが割合は低下中 — 分配に注意
  • 25%以下またはリテーラー支配の列： 弱い参加者が動きを主導 — 反転リスクが高い
  • 赤い売り列： モメンタムが反対側で50%の閾値を超えると現れる

品質フィルター

すべての列に対してアクションを取る価値があるわけではありません。AIO Banker は弱くノイジーなシグナルを抑制するために最小範囲フィルター（デフォルト：買いと売りの両方で5）を適用します。さらに、週末の暗号資産取引量は機関投資家の読み取りにとって通常信頼性が低いため、土曜日のシグナルは自動的に非表示になります。

Chaikin ボラティリティ：エントリータイミング

市場を動かしているのが誰かを知ることが方程式の半分です。もう半分はいつエントリーするかです。AIO Banker は高値-安値のスプレッドの変化率を測定する Chaikin ボラティリティオシレーターを統合しています：

Chaikin Volatility = ROC(EMA(High - Low, Length), ROC_Length)

このオシレーターがゼロを上回ると、ボラティリティが拡大していることを意味します — 新しい方向性のある動きが始まっているサインです。重要なコンフルエンス：

  • バンカー列の存在 + ボラティリティのクロスアップ = 高確率エントリー
  • ボラティリティの拡大は、機関投資家のモメンタムが静かな買い集めだけでなく、実際の価格動向に変換されていることを確認します

オプションのボラティリティフィルター

3つのオプションの確認フィルターが偽のボラティリティシグナルを防ぎます：

  1. 出来高確認： 出来高は SMA × 乗数を超える必要があります（例：20バー平均の150%）
  2. MA 以上の価格： 強気シグナルには価格が設定可能な MA（例：SMA 50）を上回る必要があります
  3. ADX トレンド強度： ADX は閾値（例：>20）を超える必要があり、市場がトレンド中でレンジではないことを確認します

KDJ 指標：トリプル確認

3番目のシグナル層は KDJ 指標 — アジア市場で人気のあるストキャスティクスの変種で、極端な状態のために “J ライン” を追加します：

  • K = スムーズ化 RSV（ランダム強度値）
  • D = スムーズ化 K
  • J = 3K − 2D（極値を検出する増幅オシレーター）

J が長いしきい値を上回り、バンカーが緑の場合、モメンタム + タイミング + 極値のトリプルコンフルエンス — 最高確率のエントリー条件が揃います。

内蔵 Magic Band 統合

AIO Banker には、プルバックゾーンのコンフルエンス専用の組み込み Magic Band（フィボナッチレベルを持つ修正 ATR スーパートレンド）が含まれています。バンカーモメンタムが活発で価格が Magic Band の 61.8% フィボナッチゾーンに引き戻される場合、内蔵の損切り参照（バンド自体）を持つ追加エントリー機会が生まれます。

CVD モード：累積ボリュームデルタ

バンカーモメンタムを超えて、インジケーターは10のイントラバー精度レベルを持つ専用 CVD（累積ボリュームデルタ）表示モードを提供します。これは request.security_lower_tf を介して低い時間軸データを使用して、バイヤー主導とセラー主導にボリュームを分解します。

CVD リセットは以下にアンカーするよう設定できます：

  • 段階的な上位時間軸（自動）
  • 固定時間軸（例：日足）
  • 固定時間（例：NY オープン 9:30）
  • レギュラーセッション開始
  • トレンド変化（スーパートレンド、アルーン、またはパラボリック SAR）

バンカー + CVD 組み合わせモードは、バンカーモメンタムと CVD の両方が方向性に合意した場合のみ列を表示します — 究極の機関投資家確認フィルター。

シグナル階層 & アラートシステム

3つのシグナルソース（バンカー、ボラティリティ、KDJ）と Magic Band 統合により、AIO Banker はアラートスパムを防ぐ階層的アラートシステムを使用しています：

  1. カラー変更アラート（最高優先度）：バンカーが赤から緑または逆に切り替わる
  2. コンフルエンスシグナル： Banker+Vol、Banker+KDJ、Banker+Vol+KDJ、またはすべての組み合わせとして設定可能
  3. Magic Band アラート： トレンド方向でのバンドタッチ
  4. 列表示アラート（最低優先度）：任意のバンカー列が表示される

実践的なトレーディングセットアップ

“バンカー + ボラティリティ + Magic Band” エントリー

  1. バンカー緑列を識別 — 機関投資家の参加が確認
  2. Chaikin ボラティリティのゼロクロスを待つ — 拡大開始
  3. 価格は Magic Band の 61.8% プルバックゾーン付近にあるべき — 最適エントリーポイント
  4. Magic Band トレイル以下にストップを置いてエントリー
  5. オプション： 追加確認のため KDJ J ラインが閾値以上であることを要求

分配検出

  • 価格が新高値を更新する中でバンカー列の高さが連続バーにわたって低下 = モメンタムダイバージェンス
  • バンカー支配からリテーラー支配への移行 = スマートマネーが退場し、リテールが参入
  • 長期間のバンカー緑の後に赤い売り列が現れる = 分配フェーズの開始

なぜこれがあなたのトレーディングに重要なのか

従来のモメンタム指標（RSI、MACD、ストキャスティクス）はすべての市場参加者を単一の数字に統合します。「強気 RSI」が協調した機関投資家の買いによるものか、CNBC ヘッドライン後のリテールのパニック買いによるものかを判断することはできません。AIO Banker Momentum は参加者層別にモメンタムを分解する唯一の TradingView インジケーターであり、スマートマネーとトレードを一致させるために必要なコンテキストを提供します。
